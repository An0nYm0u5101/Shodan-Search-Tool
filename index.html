Introduction to The Shodan Search Tool

The Shodan Search Tool assists users by specially crafting the search query based on minimal user input.

Users are able to:

  • Discover all publically available devices on their network
  • Identify vulnerabilities specific to each host
  • Search for devices on their network by port, software type, HTTP status code, vulnerability or keyword


Not quite sure what Shodan is??

Quite simply, Shodan is the search engine for the internet of things (IoT)

Unlike Google, which crawls for data on web pages and then creates an index of web content, Shodan mainly looks for ports and then grabs the resulting banners and indexes them. Specifically, Shodan uses distributed scanners throughout the world to randomly select target IP addresses and identify listening TCP and UDP ports. Listening ports are further enumerated to gather protocol banners, web pages, and other service data. All of this data is then added to an enormous, searchable database that describes the "what" of Internet devices.



This product uses the Shodan API but is not endorsed or certified by Shodan.
